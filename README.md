
# Automatizaciones - n8n

En esta carpeta se encuentran flujos y proyectos creados para **practicar y experimentar con n8n**.  
El objetivo es aprender a crear workflows, integrar APIs, procesar datos y automatizar tareas repetitivas de manera sencilla.

---

## Contenido

- Workflows de ejemplo para distintos escenarios.  
- Integración con bases de datos y servicios externos.  
- Automatización de tareas programadas y eventos disparadores.  
- Manejo básico de errores y notificaciones.  

---

## Propósito

Este repositorio sirve como **espacio de práctica y experimentación**, donde se prueban distintas funcionalidades de n8n y se desarrollan ideas de automatización sin depender de casos de uso específicos.
